给你一个字符串 s，由若干单词组成，单词前后用一些空格字符隔开。返回字符串中 最后一个 单词的长度。

单词 是指仅由字母组成、不包含任何空格字符的最大子字符串。

思路一：模拟题意
int lengthOfLastWord(char * s){
int ans = 0;
for(int i = strlen(s)-1;i>=0;i–)
{
if(s[i]!=’ ‘)ans++;
if(ans!=0&&s[i]==’ ‘)break;
}
return ans;
}

分析：
本题要求最后一个单词长度，只需从后向前遍历，ans不断增加，一旦遇到空格则输出ans的值

总结：
本题考察对字符串的应用，理解题意后可快速解决。
