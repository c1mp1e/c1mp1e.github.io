leetcode做题笔记77组合

Created2023-08-13|Updated2023-08-13
|Post View:

给定两个整数 n 和 k，返回范围 [1, n] 中所有可能的 k 个数的组合。

你可以按 任何顺序 返回答案。

思路一：直接求出组合数将每个组合放进数组中
int** combine(int n, int k, int* returnSize, int** returnColumnSizes)
{
int size = 0, num = 1, i;
int * arr = (int*)malloc(sizeof(int) * k);
assert(arr);
for (i = 0; i < k; i++)
*(arr + i) = i + 1;
for (i = 0; i < k; i++)
num = num * (n - i) / (i + 1);

        int ** c = (int**)malloc(sizeof(int*)* num);
        assert(c);
        *returnColumnSizes = (int*)malloc(sizeof(int) * num);
        assert(*returnColumnSizes);
        *returnSize = num;

        while (size < num)
        {
            *(*returnColumnSizes + size) = k;
            *(c + size) = (int*)malloc(sizeof(int) * k);
            assert(*(c + size));

            for (i = 0; i < k; i++)
                *(*(c + size) + i) = *(arr + i);
            
            for (i = k - 1; i >= 0; i--)
            {
                if (*(arr + i) < n - k + i + 1)
                    break;
            }
            if (i < 0)
                break;
            else
            {
                *(arr + i++) += 1;
                for (; i < k; i++)
                    *(arr + i) = *(arr + i - 1) + 1;
            }
            size++;
        }

        return c;
    }

分析：
本题与组合相关，直接先算出共有多少组合数，暴力列举各种可能的情况，再将组合放入数组，最后输出答案

总结：
本题考察数组的应用，实际编写时注意考虑位置是否为最大数，若不是则需重新安排，
