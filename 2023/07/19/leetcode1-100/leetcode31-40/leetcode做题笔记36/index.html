请你判断一个 9 x 9 的数独是否有效。只需要 根据以下规则 ，验证已经填入的数字是否有效即可。

数字 1-9 在每一行只能出现一次。
数字 1-9 在每一列只能出现一次。
数字 1-9 在每一个以粗实线分隔的 3x3 宫内只能出现一次。（请参考示例图）

注意：

一个有效的数独（部分已被填充）不一定是可解的。
只需要根据以上规则，验证已经填入的数字是否有效即可。
空白格用 ‘.’ 表示。

思路一：模仿题意
先判断行内数字是否重复，再判断列数字，最后判断3x3方格内是否重复

    bool isValidSudoku(char** board, int boardSize, int* boardColSize){
        int count = 0;
        int *check = (int*)malloc(sizeof(int)*10);
        for(int i = 0;i<9;i++)//判断行
        {
            for(int k = 0;k<=9;k++)check[k] = 0;
            for(int j = 0;j<9;j++){
                if(board[i][j]!='.'){check[board[i][j]-48]++;if(check[board[i][j]-48]>1)return false;}
            }    
        }
        for(int i = 0;i<9;i++)//判断列
        {
            for(int k = 0;k<=9;k++)check[k] = 0;
            for(int j = 0;j<9;j++){
                if(board[j][i]!='.'){check[board[j][i]-48]++;if(check[board[j][i]-48]>1)return false;}
            }    
        }
        for(int i = 0;i<7;i+=3)//判断3x3方格
        {
            for(int k = 0;k<=9;k++)check[k] = 0;
            for(int j = 0;j<9;j++){
                if(count%3==0)for(int k = 0;k<=9;k++)check[k] = 0;
                if(board[i][j]!='.'){check[board[i][j]-48]++;if(check[board[i][j]-48]>1)return false;}
                if(board[i+1][j]!='.'){check[board[i+1][j]-48]++;if(check[board[i+1][j]-48]>1)return false;}
                if(board[i+2][j]!='.'){check[board[i+2][j]-48]++;if(check[board[i+2][j]-48]>1)return false;}
                count++;
            }    
        }
        free(check);
        return true;
    
    }

时间复杂度O(n^2)，空间复杂度O(1)

分析：
题目要求行内1-9各数字只能出现一次，列，3x3的方格也是一样。利用数组的存放特性，将读取到的数设作数组下标，对该下标的数加一，便可判断是否有重复的数。判断3x3方格时需设置一个边界值，当前三行遍历完后需将数组清零后转换到下三行进行判断。

收获：
锻炼了拆分问题的能力，即将问题转换为三个需要解决的问题：1.判断行是否有重复数2.判断列是否有重复数3.判断3x3方格是否有重复数

该题本质上还是数组的应用，将字符串内数处理后便可得到答案。
