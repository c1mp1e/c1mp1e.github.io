第一题
给你一个下标从 1 开始、长度为 n 的整数数组 nums 。

对 nums 中的元素 nums[i] 而言，如果 n 能够被 i 整除，即 n % i == 0 ，则认为 num[i] 是一个 特殊元素 。

返回 nums 中所有 特殊元素 的 平方和 。

    int sumOfSquares(int* nums, int numsSize){
        int res = 0;
        for(int i = 0;i<numsSize;i++)
        {
            if(numsSize%(i+1)==0)
            res+=nums[i]*nums[i];
        }
        return res;
    }

根据题意作答，直接返回特殊元素平方和

第二题
给你一个下标从 0 开始的整数数组 nums 和一个 非负 整数 k 。

在一步操作中，你可以执行下述指令：

在范围 [0, nums.length - 1] 中选择一个 此前没有选过 的下标 i 。
将 nums[i] 替换为范围 [nums[i] - k, nums[i] + k] 内的任一整数。
数组的 美丽值 定义为数组中由相等元素组成的最长子序列的长度。

对数组 nums 执行上述操作任意次后，返回数组可能取得的 最大 美丽值。

注意：你 只 能对每个下标执行 一次 此操作。

数组的 子序列 定义是：经由原数组删除一些元素（也可能不删除）得到的一个新数组，且在此过程中剩余元素的顺序不发生改变。

    int cmp(const void * a,const void * b)
        {
            return (*(int*)a-*(int*)b);
        }
    
    
    int maximumBeauty(int* nums, int numsSize, int k){
        qsort(nums,numsSize,sizeof(int),cmp);
        int low = 0,high,res = 0;
        for(high = 0;high<numsSize;high++)
        {
            if(nums[high]-nums[low]>2*k)low++;
            res = res<high-low+1?high-low+1:res;
        }
        return res;
    }

排序后用双指针的方法，得到最大美丽值

第三题
如果元素 x 在长度为 m 的整数数组 arr 中满足 freq(x) * 2 > m ，那么我们称 x 是 支配元素 。其中 freq(x) 是 x 在数组 arr 中出现的次数。注意，根据这个定义，数组 arr 最多 只会有 一个 支配元素。

给你一个下标从 0 开始长度为 n 的整数数组 nums ，数据保证它含有一个支配元素。

你需要在下标 i 处将 nums 分割成两个数组 nums[0, …, i] 和 nums[i + 1, …, n - 1] ，如果一个分割满足以下条件，我们称它是 合法 的：

0 <= i < n - 1
nums[0, …, i] 和 nums[i + 1, …, n - 1] 的支配元素相同。
这里， nums[i, …, j] 表示 nums 的一个子数组，它开始于下标 i ，结束于下标 j ，两个端点都包含在子数组内。特别地，如果 j < i ，那么 nums[i, …, j] 表示一个空数组。

请你返回一个 合法分割 的 最小 下标。如果合法分割不存在，返回 -1 。

    int cmp(const void*a,const void*b)
    {
        return (*(int*)a-*(int*)b);
    }
    
    int minimumIndex(int* nums, int numsSize){
        int left = 0,tag,res,flag = 1;
        if(numsSize==1)return -1;
        int a[100000] = {0};
        for(int i = 0;i<numsSize;i++)
        {
            a[i] = nums[i];
        }
        qsort(a,numsSize,sizeof(int),cmp);
        for(int i = 0;i<numsSize-1;i++)
        {
            if(a[i]==a[i+1])flag++;
            if(flag*2>numsSize){tag=a[i];break;}
        }
            
            for(int i = 0;2*i<=left;left++)
            {
                if(nums[left]==tag)i++;
                res = left;
            }
            
            for(int k = 0;left<numsSize;left++)
            {
                if(nums[left]==tag)k++;
                if(k>=(numsSize-res+1)/2)return res;
            }
            return -1;
    
    }

先找到支配元素，此处使用排序后对相同数计数的方法，接着找到有支配数且支配数为目标数的第一个数组，在判断接下来的数组支配数是否为目标数，若是则输出分割的下标，若否则输出-1

第四题
给你一个字符串 word 和一个字符串数组 forbidden 。

如果一个字符串不包含 forbidden 中的任何字符串，我们称这个字符串是 合法 的。

请你返回字符串 word 的一个 最长合法子字符串 的长度。

子字符串 指的是一个字符串中一段连续的字符，它可以为空。

    typedef struct LetterTree{
        struct LetterTree* next[26];
        bool tail;
    }tree;
    int longestValidSubstring(char * word, char ** forbidden, int forbiddenSize){
    
        tree *root=(tree*)malloc(sizeof(tree)),*p,*q;
        root->tail=false;
        memset(root->next,0,sizeof(tree*)*26);//根节点
        
        for(int i=0;i<forbiddenSize;i++){
            p=root;
            for(int j=0;forbidden[i][j];j++){
                if(p->next[forbidden[i][j]-'a']==NULL){
                    q=(tree*)malloc(sizeof(tree));
                    q->tail=false;
                    memset(q->next,0,sizeof(tree*)*26);
                    p->next[forbidden[i][j]-'a']=q;
                }
                p=p->next[forbidden[i][j]-'a'];
                if(forbidden[i][j+1]=='\0')p->tail=true;
            }
        }
        int n=strlen(word);
        int max=0,a=0,r;
        for(int i=n-1;i>=0;i--){
            r=0;
            p=root;
            a++;
            while(i+r<n&&p!=NULL&&p->tail==false){
                p=p->next[word[i+r]-'a'];
                r++;
            }
            r--;
            if(i+r<n&&p!=NULL&&p->tail==true&&r<a)a=r;
            if(a>max)max=a;}
        void freeTree(tree* root){
            for(int i=0;i<26;i++){
                if(root->next[i]!=NULL){
                    free(root->next[i]);
                }
            }
            free(root);
        }//释放字典树
        free(root);
        return max;
    }

利用字典树，将字符串不断匹配字典树中字符串，若有相同则舍去，返回最长合法字符串长度
