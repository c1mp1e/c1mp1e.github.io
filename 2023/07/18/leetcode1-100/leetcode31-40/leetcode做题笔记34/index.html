给你一个按照非递减顺序排列的整数数组 nums，和一个目标值 target。请你找出给定目标值在数组中的开始位置和结束位置。

如果数组中不存在目标值 target，返回 [-1, -1]。

你必须设计并实现时间复杂度为 O(log n) 的算法解决此问题。

思路一：二分法
该题与上一题很像，可使用二分法是时间复杂度降低到O(logn)进行作答

    int binarySearch(int* nums, int numsSize, int target, bool check) {
        int left = 0, right = numsSize - 1, ans = numsSize;
        while (left <= right) {
            int mid = (right - left) / 2 + left;//避免数据超限
            if (nums[mid] > target || (check && nums[mid] >= target)) {
                right = mid - 1;
                ans = mid;
            } else {
                left = mid + 1;
            }
        }
        return ans;
    }
    
    int* searchRange(int* nums, int numsSize, int target, int* returnSize) {
        int leftIdx = binarySearch(nums, numsSize, target, true);//确定左边界
        int rightIdx = binarySearch(nums, numsSize, target, false) - 1;//确定右边界
        int* ret = malloc(sizeof(int) * 2);
        *returnSize = 2;
        if (leftIdx <= rightIdx && rightIdx < numsSize && nums[leftIdx] == target && nums[rightIdx] == target) {
            ret[0] = leftIdx, ret[1] = rightIdx;
            return ret;
        }
        ret[0] = -1, ret[1] = -1;
        return ret;
    }

时间复杂度O(logn)，空间复杂度O(1)

做题历程：
本题与上一题类似，将左边界和右边界利用二分法找到。实际编写时一开始未考虑将左右边界的判断用一个函数来表示，后改为函数表示时代码长度大大减少，同时需注意数字超限的问题，将mid赋值为(right - left) / 2 + left。
