leetcode做题笔记143. 重排链表

Created2023-09-17|Updated2023-09-17
|Post View:

​
给定一个单链表 L 的头节点 head ，单链表 L 表示为：

L0 → L1 → … → Ln - 1 → Ln
请将其重新排列后变为：

L0 → Ln → L1 → Ln - 1 → L2 → Ln - 2 → …
不能只是单纯的改变节点内部的值，而是需要实际的进行节点交换。

思路一：两个指针相向运动记录
c语言解法

    int left;
    struct ListNode *prev;

    void solve(struct ListNode *curr, int right) {
        if (!curr) return;
        solve(curr->next, right + 1);
        if (right > left + 1) {  // 相向运动
            struct ListNode *nxt = prev->next;
            prev->next = curr;
            curr->next = nxt;
            prev = nxt;  // 注意此时前驱
            left++;  // 移动
        } else if (right == left + 1 || right == left) { 
            curr->next = NULL;
            left++;
        }
    } 

    void reorderList(struct ListNode* head){
        left = 0;
        prev = head;
        solve(head, 0);
    }

分析：
本题要将链表重新排列，由于链表的性质不方便直接将后面的节点转移到前面，可以采用双指针的方法，两个指针一起向中间运动，直到两个指针相遇，并记录路径上的节点，最后返回即可解决

总结：
本题考察链表的重排操作，利用双指针可以解决问题，双指针的移动可以用递归来解决，熟悉后即可解决
